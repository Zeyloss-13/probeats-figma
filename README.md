# probeats-figma

Projet réalisé dans le cadre d’un exercice de maquettage UI/UX.

## Objectif
Concevoir la maquette du site e-commerce Probeats en version desktop et mobile.

## Contenu du projet
- Version desktop :
  - Page Accueil
  - Page Produit
  - Page Commande
  - Page À propos
  - Page Assistance

- Version mobile :
  - Wireframes pour l’ensemble des pages

- Header et footer communs
- Styles et composants
- Prototype de navigation

## Outils utilisés
- Figma
- Librairies d’icônes (Figma Community)

## Livrable
Le fichier `.fig` est fourni dans ce repository.
